# SamuraiFightingGame

This is a fighting game in javascript.
Only works on a computer/PC


Tutorial on how to play:

Player1:
|MOVES|COMMAND|
|-----|-------|
|Move forward| key d|
|Move backward| key a|
|Jump| key w|
|Slash| key space|

Player2:
|MOVES|COMMAND|
|-----|-------|
|Move forward| Arrow ->|
|Move backward| Arrow <-|
|Jump| Arrow Up|
|Slash|Arrow Down|
